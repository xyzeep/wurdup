How to play

Guess the word in six tries by entering valid five-letter words, with the tile colors changing to indicate how close your guess is to the correct word.

The letter A is in the word and on the correct position.

The letter F is in the word but not in the correct position.

The letter I is in not in the word.

About WurdUp

Wurdup is a word-guessing game built with HTML, CSS, and JavaScript. Players guess a secret word based on feedback form each guess to find the correct answer. The code behind the game is available on GitHub and anyone is welcome to explore.
